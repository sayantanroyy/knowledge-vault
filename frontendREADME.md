```markdown
# KnowledgeVault Frontend

React/Next.js frontend for KnowledgeVault AI.

## Setup

\`\`\`bash
npm install
cp .env.example .env.local
npm run dev
\`\`\`

Visit http://localhost:3000

## Project Structure

- `components/` - React components
- `services/` - API calls
- `styles/` - CSS styling
- `public/` - Static assets

## Build

\`\`\`bash
npm run build
npm start
\`\`\`

## Deployment

Deploy to Vercel:
\`\`\`bash
vercel
\`\`\`
```
